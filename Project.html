Final Project    Video Frames Compression Using Matlab

Announcements

Mar 30,  2009    The requirement of final project is available.

General information

1. The final project of this course is to design and implement image compression techniques using Matlab.

2. The deadline of the final project is Tue Dec. 28 at 23:59.

3. The final project is the group work with 1 ~ 4 person(s) each group. The group members have the same scores for the final project.

4. Final project is 20% of the assessment of this course.

Requirements

1. Download the files below to the same local folder.

  • Image1.bmp, Image2.bmp, Image3.bmp, Image4.bmp                 test data
  • CalMSE.m                                                                                  evaluate the quality of the recovered images 
  • Compratio.m                                                                               evaluate the compression ratio of the compressed file
  • Compress.m                                                                                compress the original images to the compressed file
  • Decompress.m                                                                            decompress the compressed file to the recovered images
  • imdisplay.m                                                                                 display the image
  • main.m                                                                                        main function of the project

2. You are required to implement image compression and decompression by modifying the two functions: Compress.m and Decompress.m.

3. Submit the two functions to TA by email csygliu@comp.polyu.edu.hk before the deadline

4. Don't make any modification to the other files except Compress.m and Decompress.m.

Submission

1. Submit Compress.m and Decompress.m to TA by email: csygliu@comp.polyu.edu.hk before Tue Apr. 28 at 23:59.

2. Penalty for late submission

  • Wed Apr 29 00:00 - 23:59                   25% deduction
  • Tur Apr 30 00:00 - 23:59                     50% deduction
  • Fri May 1 00:00 - 23:59                       75% deduction
  • After May 2 23:59                                Zero point

3. Don't submit any other help functions, images and files.

grading schemes

The final project is totally 100 points. Your implementation is evaluated from three aspects:

1. 40 points for the compression ratio between the original images and the compressed file.

    If the compression ratio is equal or smaller than 1, you will get zero point for the final project.

2. 40 points for the quality of the recovered images.

    20 Points:    Objective evaluation of the image quality by calculate the MSE between the original images and the recovered images.
    20 Points:    Subjective evaluation of the image quality by the observation of the TA.   

3. 10 points for the running time of the compression and decompression algorithms.

4. 10 points for the quality of robustness of the compression and decompression techniques by evaluating on the unknown four images.

　

Friendly reminder

1. Don't miss the submission deadline.

2. The components in the matrixes orig_image, comp_image and reco_image should be the integers.

3. The size of the reco_image should be same with the size of orig_image.

4. Don't make any modification of the provided functions expect Compress.m and Decompress.m.

5. Don't read the image file when you modify Compress.m and Decompress.m, otherwise, you will get zero point in the final project.

6. The compression ratio should larger than 1, otherwise, you will get zero point in the final project.
